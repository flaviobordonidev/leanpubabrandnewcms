## How do I open multiple instances of Visual Studio Code?

For Mac:
**⌘ + Shift + N** : Open new Window
**⌘ + K** then release the keys and press **O** (open current Tab in new Window)

> For windows **Ctrl + Shift + N** will open a new window.
