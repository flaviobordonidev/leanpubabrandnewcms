# il Blog

Fla e Bob decidono che per la parte del blog utilizzeranno:
blog-small.html         per blog-index 
blog-single-small.html  per blog-show

Fla inizia a preparare il mock per il Blog. 

Sa che è sul branch master quindi attualizza git, crea nuovo branch e lo aggiorna su github


{title="terminal", lang=bash, line-numbers=off}
~~~~~~~~
$ git checkout master
$ git pull origin master

$ git checkout -b mock_post
$ git push origin mock_post
~~~~~~~~

Siccome il blog è fatto da tanti articoli (posts) Bob copia e rinomina i due files: 
  views/mocks/posts_index.html.erb
  views/mocks/posts_show.html.erb

Nei due files lascia solo il codice dentro il <body> ad eccezione della chiamata javascript a functions.

modifica il file routes

[codice: mocks config 02](#code-mocks-config-02)

{title="config/routes.rb", lang=ruby, line-numbers=on, starting-line-number=3}
~~~~~~~~
  get 'mocks/posts_index'
  get 'mocks/posts_show'
~~~~~~~~

verifica che si visualizzano su localhost:3000/mocks/index e .../show ("$ rails s" attivo)
