# Slider Revolution - Focus Parallax

Prendiamo una degli slider premium. Scegliamo "focus parallax"
Modifichiamolo caricando le nostre immagini sfruttando Slider Revolution Stand Alone


## Effettuiamo l'export su HTML

Dopo aver preparato il nostro slider usiamo l'opzione "Export to HTML"

![Fig. 01](images/04-slider-revolution/01-sr-focus_parallax/01_01-sr-export_to_html.png)

Questo scarica un file .zip nella nostra cartella download. 




## Verifichiamo lo slider

Creiamo la cartella "sr" dentro C:\MAMP\htdocs
Scompattiamo il file ".zip" e copiamolo dentro C:\MAMP\htdocs\sr
Avendo MAMP attivo (ossia Apache Server e MySQL Server) possiamo verificare il contenuto aprendolo sul browser all'url:
  http://localhost:80/sr/slider.html

Verificato che il file è quello che ci aspettavamo possiamo importarlo nella nostra app su Rails.

Il contenuto del file ".zip" lo puoi trovare qui allegato dentro la cartella **01-exported_slider** 




## Capiamo l'uso dei vari files allegati

cancellando un file alla volta ci rendiamo conto delle modifiche che apporta.
Per esempio provando a cancellare dei files ci rendiamo conto che una chiamata che permette di visualizzare il tutto è

"asset/ic_expand_more_24px.svg"

Se la eliminiamo la pagina resta bianca!
