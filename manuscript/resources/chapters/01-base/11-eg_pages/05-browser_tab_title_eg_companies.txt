# Cap 11.5 -- Rendiamo multilingua il titolo nel tab del browser per eg_companies

Attiviamo le traduzioni per il titolo nel tab del browser.




## Apriamo il branch "Browser Tab EgCompanies"

{caption: "terminal", format: bash, line-numbers: false}
```
$ git checkout -b btec
```




## Implementiamo internazionalizzazione (i18n)

Creiamo delle nuove voci nei locales che aggiungeremo poi alle views.

{id: "01-10-05_01", caption: ".../config/locales/it.yml -- codice 01", format: yaml, line-numbers: true, number-from: 4}
```
  eg_companies:
    index:
      html_head_title: "Tutte le aziende"
    show:
      html_head_title: "Az."
    edit:
      html_head_title: "Modifica"
    new:
      html_head_title: "Nuova azienda"
```



{id: "01-10-04_02", caption: ".../config/locales/en.yml -- codice 02", format: yaml, line-numbers: true, number-from: 4}
```
  eg_companies:
    index:
      html_head_title: "All companies"
    show:
      html_head_title: "Company"
    edit:
      html_head_title: "Edit"
    new:
      html_head_title: "New company"
```





## Aggiungiamo le chiamate alle views

traduciamo index

{title=".../app/views/eg_companies/index.html.erb", lang=HTML+Mako, line-numbers=on, starting-line-number=3}
```
<% provide(:html_head_title, "#{t '.html_head_title'}") %>
```

traduciamo show

{title=".../app/views/eg_companies/show.html.erb", lang=HTML+Mako, line-numbers=on, starting-line-number=3}
```
<% provide(:html_head_title, "#{t '.html_head_title'}: #{@eg_company.name}") %>
```

traduciamo edit

{title=".../app/views/eg_companies/edit.html.erb", lang=HTML+Mako, line-numbers=on, starting-line-number=3}
```
<% provide(:html_head_title, "#{t '.html_head_title'} #{@eg_post.headline}") %>
```

traduciamo new

{title=".../app/views/eg_companies/new.html.erb", lang=HTML+Mako, line-numbers=on, starting-line-number=3}
```
<% provide(:html_head_title, "#{t '.html_head_title'}") %>
```
